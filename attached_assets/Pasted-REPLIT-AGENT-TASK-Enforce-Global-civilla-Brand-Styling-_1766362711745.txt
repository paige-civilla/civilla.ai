PART B — CREATE Brand COMPONENTS
3) Create: client/src/components/Brand.tsx with:

- export function Brand({ children }: { children: string })
  -> returns <span className="civilla-brand">{children}</span>

- export function BrandText({ children }: { children: React.ReactNode })
  -> If children is a string, replace any occurrences of:
     www.civilla.ai
     civilla.ai
     civilla
     (case-insensitive)
     with <span className="civilla-brand">MATCH_LOWERCASED</span>
     and return an array of nodes.
  -> If children is not a string (e.g. JSX), just return children unchanged (don’t break layouts).

Use a regex with word boundaries so “civilla” doesn’t match inside other words:
/(\bwww\.civilla\.ai\b|\bcivilla\.ai\b|\bcivilla\b)/gi

When rendering a match, output it as match.toLowerCase().
